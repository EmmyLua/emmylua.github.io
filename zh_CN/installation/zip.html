从zip文件安装

  • 下载zip文件

    • 从 JetBrains Plugin Repository 下载

    • 从 Github release page 下载(可能不是最新版本)

    • 从QQ群文件下载(最新版首先在群内发布测试，群号：29850775 )

    警告

    无需解压Zip文件

  • 选择菜单 File -> Settings -> Plugins -> Install plugin from disk

  • 选择刚下载的EmmyLua.zip文件，安装完毕重启IDEA
